Descripción detallada

Clase que implementa una interfaz de aplicación para controlar 6 ejes.

La API está formada por la función exec() que debe ser llamada en un proceso a ejecutarse en tiempo real y por método que interpreta comandos a ejecutar llamada request(). Para más información sobre los comandos (ver Comandos de API).
